Introduction

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It is the foundation of web development and is used to structure content on the web.

What you should already know

This guide assumes you have the following basic background:

  • A general understanding of the Internet and the World Wide Web (WWW).
  • Some familiarity with using a text editor and file management.
  • Basic understanding of web browsers and how they render HTML.
Hello world
To get started with writing HTML, open a text editor and write your first "Hello world" HTML code: 

<!DOCTYPE html>
<html>
  <head>
    <title>Hello World</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>
Elements and Tags
HTML documents are made up of elements, which consist of tags. Tags are used to mark up the content of the document. For example, the <p> tag defines a paragraph, and the <a> tag defines a hyperlink.
Basic HTML Structure

A basic HTML document has the following structure:


            <!DOCTYPE html>
            <html>
              <head>
                <meta charset="UTF-8">
                <title>Document Title</title>
              </head>
              <body>
                <!-- Content goes here -->
              </body>
            </html>
Headings

HTML provides six levels of headings, from <h1> to <h6>. The <h1> tag defines the most important heading, and the <h6> tag defines the least important heading.

  • <h1> - The main heading, usually the largest. It is typically used for the title of the page.
    • Example: <h1>This is a Level 1 Heading</h1>
  • <h2> - Secondary heading, used for major sections of content.
    • Example: <h2>This is a Level 2 Heading</h2>
  • <h3> - Tertiary heading, used for subsections under <h2> sections.
    • Example: <h3>This is a Level 3 Heading</h3>
  • <h4> - Fourth-level heading, used for subsections under <h3> sections.
    • Example: <h4>This is a Level 4 Heading</h4>
  • <h5> - Fifth-level heading, used for even smaller sections.
    • Example: <h5>This is a Level 5 Heading</h5>
  • <h6> - Sixth-level heading, the smallest heading level.
    • Example: <h6>This is a Level 6 Heading</h6>

            <h1>Main Heading</h1>
            <h2>Subheading</h2>
            <h3>Sub-subheading</h3>
            <!-- and so on -->
Paragraphs

Paragraphs are defined using the <p> tag.


            <p>This is a paragraph.</p>
            <p>This is another paragraph.</p>
Links

Links are defined using the <a> tag. The href attribute specifies the URL of the linked document. 


            <a href="https://www.example.com">Visit Example</a>
Images

Images are defined using the <img> tag. The src attribute specifies the path to the image, and the alt attribute provides alternative text for the image. 


                <img src="image.jpg" alt="Description of the image">
Lists

HTML supports ordered lists <ol> and unordered lists <ul>. List items are defined using the <li>tag.

Example:

  • Item 1
  • Item 2
  • Item 3
  1. First item
  2. Second item
  3. Third item

Code 

                <ul>
                  <li>Item 1</li>
                  <li>Item 2</li>
                  <li>Item 3</li>
                </ul>
                
                <ol>
                  <li>First item</li>
                  <li>Second item</li>
                  <li>Second item</li>

Tables

Tables are defined using the <table> tag. Rows are defined using the <tr> tag, and table cells are defined using the <td> tag. Table headers are defined using the <th> tag.

Forms

Forms are used to collect user input and are defined using the <form> tag. Form elements include input fields <input>, text areas <textarea>, and buttons <button>.


                <form action="/submit" method="post">
                  <label for="name">Name:</label>
                  <input type="text" id="name" name="name">
                  <button type="submit">Submit</button>
                </form>
